Project Overview

This project is an end-to-end autonomous LinkedIn content generation and publishing system built using n8n, AI agents, Gemini LLMs, Tavily Search, Cloudflare Workers, and LinkedIn APIs.

The system acts as an autonomous AI content engine for professional LinkedIn publishing, transforming a simple user idea into a fully researched and visually enhanced LinkedIn post without manual intervention.

Zero Cost Architecture

One of the most significant achievements of this project is that the entire workflow is built without spending a single penny. I leveraged open-source tools and free-tier AI services to create a production-grade automation system.

Self-Hosted Automation

Instead of using expensive cloud automation platforms, n8n is self-hosted using Docker Desktop. To make the local instance accessible to external webhooks (like Telegram and LinkedIn), I used ngrok for secure tunneling.

Free AI Power

The core intelligence is powered by Google Gemini 2.5 Flash Lite, available for free via the Google AI Studio API.

Complete Workflow Explanation

The entire content pipeline is automated across strategic stages, from initial intake to final publishing feedback.

Step 1 — Telegram Trigger
Node: Telegram Trigger

The workflow starts when a user sends a message to a Telegram bot, which acts as the primary input interface.

Step 2 — AI Research & Content Generation
Node: AI Agent | Model: Gemini 2.5 Flash Lite | Tool: Tavily Search

The topic is researched in real-time using Tavily Search, identifying trending insights and strategic industry effects.

Step 3 — Visual Prompt Engineering
Node: AI Agent1 | Model: Gemini 2.5 Flash Lite

A specialized agent converts the post into a diffusion-optimized image generation prompt, focusing on high-fidelity cinematic visuals.

Step 4 — Serverless Image Generation
Node: HTTP Request | Platform: Cloudflare Workers

A custom serverless API on Cloudflare Workers generates high-fidelity images using Cloudflare's AI infrastructure at zero cost.

Step 5 — LinkedIn Publishing
Node: Create a post

The final content and generated media are automatically published to LinkedIn via OAuth 2.0 integration.
